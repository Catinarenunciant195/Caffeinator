## 🧭 How It Works

Caffeinator watches for apps you choose. When one of those apps is active, it tells Windows to keep the system awake.

Typical flow:
1. You open an app you want to protect from sleep.
2. Caffeinator detects it.
3. Windows keeps the screen and system awake.
4. When the app closes or loses the rule, normal sleep behavior returns.

This helps when you need predictable power behavior without turning off sleep for your whole PC.

## 🛠️ Troubleshooting

### The app does not open
- Run it again from the extracted folder.
- Right-click the app and choose Run as administrator.
- Check whether Windows Defender blocked the file.
- Make sure the download finished fully.

### I do not see the tray icon
- Click the hidden icons arrow near the clock.
- Look for the Caffeinator icon there.
- Check whether the app is still running in Task Manager.

### My PC still sleeps
- Make sure the correct app rule is added.
- Open the app that matches the rule.
- Check that no other power tool is forcing sleep.
- Confirm the app is still active in the tray.

### The screen stays awake when I do not want it to
- Open the tray menu and turn off the rule.
- Remove the app from the keep-awake list.
- Close Caffeinator if you do not need it running.
